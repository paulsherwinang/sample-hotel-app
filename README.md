# sample-hotel-app

## Prerequisite
1. node
2. npm
3. git

```
git clone https://github.com/paulsherwinang/sample-hotel-app.git
npm install
bower install
```

## Running the application
```
npm start
```
Access `http://localhost:8000`


## Running Unit Tests
```
npm test
```
